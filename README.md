# Magic Job Finder MVP

Минималистичный MVP сайта, который ищет вакансии и заказы по живому описанию человека.

## Что уже есть

- красивый лендинг с ощущением "магии"
- поиск по свободному описанию навыков и желаний
- переключение между `удаленно`, `в городе`, `не важно`
- режимы `все`, `заказы`, `full-time`
- агрегация вакансий из `hh.ru` и `superjob`
- локальный AI-style разбор промпта без внешней модели
- fallback на демо-вакансии, если внешний источник не отвечает

## Как запустить

```bash
node server.js
```

После этого открой [http://localhost:3000](http://localhost:3000).

## Важная заметка

- `hh.ru` использует публичный API и должен работать без ключа.
- `superjob` в этом MVP вызывается без обязательной авторизации для базового поиска вакансий, как это описано в их документации. Если позже понадобится стабильность для production, лучше добавить `SUPERJOB_API_KEY`.

Пример запуска с ключом:

```bash
SUPERJOB_API_KEY=your_key_here node server.js
```
